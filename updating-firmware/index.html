    Updating Firmware

    Requirements

    • Linux or Windows system
    • MacOS is not supported due to a driver issue with the USB <-> UART chip CH343p
    • There is a MacOS driver, but it is untested and the status is unknown: https://github.com/WCHSoftGroup/ch34xser_macos
    • Espressif esptool

    Steps

    1. Install esptool: pip install esptool
    2. Download the latest firmware version flash.bin
    3. Push and hold the DFU button on the device (left button) while inserting it into the computer, then release the DFU button
    4. Identify the device esptool security-info
